== Description ==

Stars SMTP Mailer Plugin that throws outgoing email via SMTP through PHP Mailer. Supports all emails like Gmail,Yahoo,Outlook,Zoho etc. Maintain Email Logs and lot more.
Major features in Stars SMTP Mailer include:
1. Add any SMTP accounts to send email from 
2. Send Emails via SMTP Account
3. Save email sent log
4. Test synced email accounts

== Installation ==
= Plugin =
1. Copy the 'StarsSMTPMailer' folder into your plugins folder
2. Activate the plugin via the Plugins admin page
